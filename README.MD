<h1 align="center">Analisador Lotofácil - Estatísticas Transparentes</h1>

<h2>Descrição</h2>
<p>O Analisador Lotofácil é uma aplicação web que fornece análises estatísticas sobre os resultados históricos da loteria Lotofácil. O projeto tem como objetivo mostrar padrões e tendências nos sorteios de forma transparente, sempre lembrando que a Lotofácil é um jogo de sorte pura e que nenhuma análise estatística pode prever números com certeza.</p>

<h2>Funcionalidades Principais</h2>
<ul>
<li>📊 Dados Históricos: Visualização do último concurso, data do sorteio e quantidade de concursos analisados.</li>

<li>🔍 Análise Estatística:</li>
<ul>
<li>Números mais atrasados (que não saem há mais tempo)</li>
<li>Pares de números mais frequentes e menos frequentes</li>
<li>Gráficos de tendências</li>
</ul>
<li>🎯Sugestões de Jogos: Geração de jogos baseados em estatísticas (apenas para fins educacionais)</li>
<li>🎰Volante Interativo: Visualização dos números atrasados e sugeridos</li>
</ul>

<h2>Tecnologias Utilizadas</h2>
